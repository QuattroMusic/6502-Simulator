# 6502-Simulator
A simple to use 6502 emulator / simulator, featuring a code viewer, registers viewers
and an input display!

### Specifications
- 32kB RAM and ROM
- 120x80 display
- date and time in RAM
- keyboard input
- frequency selector in the range of 1Hz to 1GHz
